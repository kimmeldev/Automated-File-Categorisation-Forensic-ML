# Automated File Categorisation for Forensic Investigators

## Overview
This project is a web-based digital forensic system that automates file categorisation and detects suspicious files using machine learning.

## Features
- File classification using Random Forest
- Suspicious file detection (entropy, mismatch, confidence)
- Case management system
- Report generation (CSV)

## Technologies
- Python (Flask)
- Scikit-learn
- Firebase Firestore

## How to Run
pip install -r requirements.txt
python app.py

## Note
Sensitive files such as Firebase credentials and datasets are excluded for security purposes.
